This is a demo to show you using aciTree with checkboxes. The isCheckbox flag need to be set for each item into the JSON data when both checkbox and radio extensions are included.
By default, all items will have a checkbox (if isCheckbox is missing from the JSON data).

When a checkbox is checked, all childs from each level under the selected one will be checked by default. Also, the parents will be checked all the way up to the root item.
This functionality will break once there is a missing checkbox. In this way you can form checkbox groups.

Note: When loading nodes, the checked/unchecked state need to be correctly set in the JSON data.
